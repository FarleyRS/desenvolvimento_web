Sobre o HTML

HTML básico

HTML (Linguagem de Marcação de Hipertexto) é o código que você usa para estruturar uma página web e seu conteúdo. Por exemplo, o conteúdo pode ser estruturado em parágrafos, em uma lista com marcadores ou usando imagens e tabelas. Como o título sugere, este artigo fornecerá uma compreensão básica do HTML e suas funções.

Para criarmo uma pagina simples em html devemos especificar qual a versão que vamos utilizar, por exemplo esse site foi criado em HTML5. Mas como indentificamos a versão que o site foi criado, é simples todo site estruturado em HTML5 possui em seu código a teg <!DOCTYPE HTML>.

Agora podemos conhecer a estrutura completa de um documento html.
